ou usar a tag (video) com a tag (source).

lembrar que é bom usar isso para abrir em diversos navegador, a ordem é prioritaria do menor para o maior sempre! assim como nas imagens também!

após tag video podemos colocar tag (width) para tamanho e poster para capa e (controls) para ligar controle, (autoplay) para auto play no video, (loop) para recomeçar assim por diante.
